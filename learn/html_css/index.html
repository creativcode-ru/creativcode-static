1-я часть. Начинаем с нуля

Вы будете получать домашние задания, а на практических занятиях вы закрепите полученные знания. Вы начнете работать в профессиональной среде разработки Visual Studio и разместите свой рабочий проект в облачном репозитарии.

2-я часть., Уже есть базовые навыки
